# Itzfizz Scroll-Driven Hero Section Animation

This project is created as part of the Web Development Internship assignment for Itzfizz Digital.

## 🚀 Project Overview
The goal of this project is to recreate a scroll-driven hero section animation inspired by the provided reference. The project demonstrates smooth UI animations, scroll interactions, and responsive layout.

 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- GSAP (GreenSock Animation Platform)
- Bootstrap

## ✨ Features
- Full-screen hero section
- Letter-spaced animated headline
- Animated statistics cards
- Scroll-driven hero image animation
- Smooth GSAP animations
- Scroll progress indicator
- Responsive design (mobile + desktop)
